## Instructions for Group Members

1. Clone the repository:
   ```bash
   git clone https://github.com/sarasithagalagama/vehicle-service-system.git
   cd vehicle-service-system
   ```

2. Switch to your assigned branch:
   ```bash
   git checkout [your-branch-name]
   ```

3. Add your components and push:
   ```bash
   git add .
   git commit -m "Add [your component] - [your name]"
   git push origin [your-branch-name]
   ```
